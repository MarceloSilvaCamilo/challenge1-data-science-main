# Challenge Data Science - Alura Store Brasil 📊

Este projeto faz parte do **Challenge BI & Data Science da Oracle Next Education (ONE)** em parceria com a Alura. O objetivo é realizar uma análise exploratória de dados de vendas de quatro lojas distintas para auxiliar na tomada de decisão estratégica sobre faturamento e desempenho.



## 📌 Contexto do Projeto
A "Alura Store Brasil" possui quatro unidades e precisa entender qual delas apresenta o melhor desempenho e qual pode precisar de ajustes estratégicos. Através do Python e da biblioteca Pandas, extraí insights sobre faturamento total, ticket médio e categorias de produtos mais vendidas.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3
* **Bibliotecas Principais:** * `Pandas`: Manipulação, limpeza e tratamento de dados.
    * `Matplotlib`: Visualização de dados e criação de gráficos.
* **Ambiente:** Google Colab

## 📂 Estrutura do Repositório
* `AluraStoreBrasil.ipynb`: Notebook com todo o código de importação, tratamento e visualização.
* `base-de-dados-challenge-1/`: Pasta contendo os arquivos CSV das 4 lojas analisadas.

## 📈 Insights da Análise
Durante o desenvolvimento, o foco foi responder a questões críticas de negócio:
1.  **Liderança de Vendas:** Comparação de faturamento bruto entre as unidades.
2.  **Mix de Produtos:** Análise da distribuição de categorias (eletrônicos, móveis, etc.) nas lojas com menor faturamento.
3.  **Experiência do Cliente:** Avaliação média por unidade para medir a satisfação.
4.  **Logística:** Impacto do frete médio no custo final das operações.
